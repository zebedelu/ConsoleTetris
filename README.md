## Sobre o projeto

Esse projeto foi desenvolvido **mais por diversão**, sem nenhum objetivo prático ou profissional. A ideia surgiu como um desafio de recriar um clássico dos jogos em um ambiente tão simples quanto o terminal.

Tudo foi feito **em apenas 1 dia**, sem utilizar nenhuma ferramenta de IA, apenas código escrito na mão, muito café e nostalgia das horas de tetris jogadas.

## Tecnologias

- **Python 3** puro
- Sem bibliotecas extras

## Como jogar

Clone o projeto:
```bash
git clone https://github.com/zebedelu/ConsoleTetris.git
cd ConsoleTetris
```

Execute o jogo:
```bash
python main.py
```

### Controles
Use as teclas:

- Setas esquerda/direita → mover a peça

- Seta para baixo → descer a peça mais rápido

- Clicar Espaço → descer de uma vez

## Estrutura do projeto

- main.py — Código para iniciar o jogo
- game.py — Toda a lógica do Tetris
- configs.py — Configurações (Altere caso achar necessário)
- pieces/ — Objetos do Tetris (peças)
- scripts/ — Scripts essenciais
